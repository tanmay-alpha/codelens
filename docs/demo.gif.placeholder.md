# Demo media

This directory is reserved for demo media used in the top-level
[README](../README.md).

| File        | Purpose                                                     |
| ----------- | ----------------------------------------------------------- |
| `demo.gif`  | Animated walkthrough of the dashboard, VS Code extension, and PR review page. |
| `demo.mp4`  | Higher-resolution alternative.                              |
| `logo.svg`  | Project logo (eye icon).                                    |

Drop the actual `demo.gif` here and the README's `![Demo](docs/demo.gif)`
reference will resolve automatically. (We do not commit binary media to
keep the repo lean — host the GIF on the GitHub release page or
[docs.codelens.dev](https://docs.codelens.dev) and update the README link
accordingly.)
